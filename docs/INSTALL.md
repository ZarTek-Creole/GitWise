# Installation

## Prerequisites

- Python 3.8 or above
- git installed on your machine

## Installation Steps

1. Clone the repository:
```bash
git clone [repository-url]
```

2. Navigate to the cloned directory:
```bash
cd gitwise
```

3. Install the required dependencies:
```bash
pip install -r requirements.txt
```

4. Run GitWise:
```bash
python -m src.main
```
